# Movie Booking Platform

## Overview
Backend system for browsing shows and booking movie tickets.

## Features
- Browse shows by movie, city, and date
- Book tickets with seat selection
- Transactional seat locking

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## APIs

### GET /shows
Browse shows

### POST /bookings
Book tickets

## How to Run
mvn spring-boot:run
